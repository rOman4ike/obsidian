### What I’m trying to do

Set a default location within my vault for where clippings are to be placed.

### Things I have tried

Looked in help and can’t find it.

I tried entering in a default vault with vault\_name/01\_Inbox/ which is the name of my inbox folder. However when I tried to clip something I got an error message vault not found.

Then I tried a fully qualified path name from Users all the way down to the inbox folder and got the same error.

I also tried without the ending /

Then I tried putting the 01\_Inbox/Clippings by typing over the Clippings folder and that also failed.

I do not want a new folder created in my vault. I want the clippings folder located in my inbox folder.

How do I set that up?

## post by anon80950206 on Feb 5, 2025

I *believe* the only way to do this is through the templates in the Web Clipper settings in your browser.

[![folder](https://forum.obsidian.md/uploads/default/optimized/3X/4/b/4bb3ca8377f7fd0c4e11432e4515147cfc645dfd_2_580x500.jpeg)](https://forum.obsidian.md/uploads/default/original/3X/4/b/4bb3ca8377f7fd0c4e11432e4515147cfc645dfd.jpeg "folder")

I just changed the template above to `01_Inbox/Clippings` and the clipped note went to that folder. Not what you need?

## post by DiCaver on Feb 6, 2025

[DiCaver](https://forum.obsidian.md/u/dicaver) VIP

[Feb 2025](https://forum.obsidian.md/t/obsidian-web-clipper-set-default-folder/96186/3 "Post date")

In the General settings, first, you need to define Vault(s), and then, as [@anon80950206](https://forum.obsidian.md/u/anon80950206) pointed out, you define just the folder without Vault. Vault, you select from the dropdown below the Note location—path to the folder you specify on Template(s).
