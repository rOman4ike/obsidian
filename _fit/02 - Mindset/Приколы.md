# Конструкции

### Редупликация + Рифмованная пара

1. Формула: `X → X'` (меняется первая согласная, можно наоборот)
2. Есть «любимые» буквы для таких слов: **б, м, ш, л, п**
3. Используй «шм-» для пародии: деньги-**шменьги**, код-**шмод**


Примеры:
1. Примитивное: Тумба-юмба, Унга-бунга Чунга-чанга, Умпа-лумпа, бум-бум, тяп-ляп
