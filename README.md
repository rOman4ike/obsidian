Настройка `Obsidian` позволила решить следующие проблемы:
1. Сохранение всего пройденного материала в одном хранилище
2. Хранилище может легко передаваться между несколькими устройствами
3. Быстрое создание заметок (Daily Notes)

Структура проекта + большинство идей взяты из [видео](https://www.youtube.com/watch?v=rAkerV8rlow&t=419s)

## Используемые инструменты и плагины

Плагины для Obsidian:
1. `Advanced Tables`
2. `Excalidraw`
3. `Git`
4. `Iconize`

Для MacOs:
1. `Obsidian Web Clipper`
2. `Shortcuts`
3. `Codex`

Для Iphone:
1. `GitSync`
2. `Shortcuts`
