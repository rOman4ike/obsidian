# Start

Настройка `Obsidian` позволила решить следующие проблемы:
1. Сохранение всего пройденного материала в одном хранилище
2. Хранилище может легко передаваться между несколькими устройствами
3. Быстрое создание заметок (Daily Notes)
4. ...

Большинство идей взяты из [видео 1](https://www.youtube.com/watch?v=rAkerV8rlow&t=419s) и [видео 2](https://www.youtube.com/watch?v=YNcbFwp9tNU)

# Tags

Разделение заметок происходит по папкам, из-за этого использование тегов сильной пользы не приносит, однако есть некоторые теги которые используются в работе:

State tags:
- `state/initial` - заметка создана, но контента нет;
- `state/raw` - в заметке лежит сырой материал, ссылки, цитаты, наброски;
- `state/processing` - идет нормализация: структура, формулировки, чистка;
- `state/completed` - заметка полностью готова.
